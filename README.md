# Assignment-Day-21
github link : https://github.com/omshinde-alt07/Assignment-Day-21
## Overview

This project contains two Jupyter Notebook files:

* **am.ipynb** → Contains Part A (Concept Application)
* **pm.ipynb** → Contains Part B and Part C (Advanced Concepts and Theory)

The assignment covers NumPy operations, regression vs classification, and bias-variance tradeoff with practical examples.

---

## Contents

### am.ipynb

* NumPy array creation (1D, 2D, 3D)
* Indexing and slicing
* Element-wise operations
* Broadcasting
* Vectorized operations
* Basic dataset analysis

### pm.ipynb

* Matrix operations (multiplication, transpose, determinant)
* Solving linear equations
* Performance comparison (NumPy vs loops)
* Regression vs Classification
* Bias-Variance Tradeoff
* Model training and visualization
